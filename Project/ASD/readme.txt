# 요구 사항

시간 설정 기능
   요일에 상관없이 ON/OFF 시간 설정
요일 설정 기능
   요일에 따른 ON/OFF 시간 설정
자동문 열림 기능
  - NO 접점이 ON 되어 있으면 계속 열림
자동문 닫힘 기능
  - NO 접점이 off 되어 있는 상태에서 나가는 버튼 신호가 
    안들어오게 하면 계속 닫힘


# 제어시나리오

원격 제어 시
   - NC 접점을 0.5초 트리거 신호로 도어 Open
   - Exit 버튼으로 제어 가능
   - PIR 센서로 제어 가능

락 모드 시 (내부 Exit 버튼 및 PIR 센서 사용 안됨.)
  - NO 접점이 OFF 됨.
  - PIR 센서 및 Exit 버튼 동작 안함.

완전개방
  - 완전개방 버튼을 누르면 NC 접점이 ON이 계속됨.

타이머 기능
  - 시간 설정 기능 : 요일에 상관없이 시간 설정으로 ON/OFF
  - 요일 설정 기능 : 요일 및 시간설정으로 ON/OFF

# 제약사항
